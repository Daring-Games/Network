LAN

LAN's full form is Local Area Network. it connects computers within a limited area, such as in a campus, school, residence etc. A LAN may be large or small. It can extend from one user to thousands of users.

Comparing with WAN though, WAN or Wide Are Network (see WAN), has more range than LAN, and MAN or Metropolitan Area Network (see MAN) covers a network range for a whole city.

ADVANTAGES

Sharing Resources

All the data and the resources are attached to one network and all the other computers atached to that network can access that data

Communication Is Easy

In LAN, computers can exchange data very easily and fast. This saves our time and also does not affect a computer's working speed. Every computer user can share messages and data with other user on LAN. The user can log in from any computer and can access that data because computers are interconnected by a single network.

DISADVANTAGES

Security Issues

In LAN, if the server computer is not set up correctly then there is a leak in security and hackers can steal the data. So, we must set up the servers correctly and apply security protocols.

Server Crashes Affects All Computers

If there is any file, folder on the server that is corrupted then all computers attached to that server via LAN network face problems.
